# 1-2 Worker Node
## 1-2-1 역할
  - 애플리케이션의 구성요소인 파드를 호스트한다
  - 동작중이 파드를 유지
  - 쿠버네티스 런타임 환경을 제공   
    

   
## 1-2-2 구성 요소   
![image](https://github.com/JunPyo0117/CI-CD/assets/71053769/06e69901-7135-43d3-8e78-55ba5f1bad31)
  - **Kublet**
    - 파트에서 컨테이너가 동작하도록 관리
  - **Kube Proxy**
    - 클러스터의 각 노드에서 실행되는 네트워크 프록시
    - 쿠버네티스의 서비스 개념을 구현
    - 노드의 네트워크 규칙을 유지관리
